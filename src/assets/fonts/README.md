# Fonts Directory

## Libre Caslon Text

Download Libre Caslon Text font files from:
- Google Fonts: https://fonts.google.com/specimen/Libre+Caslon+Text
- Or directly: https://github.com/google/fonts/tree/main/ofl/librecaslontext

Place the following files in this directory:
- `LibreCaslonText-Regular.woff2`
- `LibreCaslonText-Italic.woff2`
- `LibreCaslonText-Bold.woff2` (optional)

You can download the TTF files and convert them to WOFF2 format using:
- Online: https://cloudconvert.com/ttf-to-woff2
- Or use the `woff2_compress` tool locally

This font serves as a fallback when Adobe Typekit is unavailable.
